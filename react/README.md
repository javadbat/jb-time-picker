# JBTimePicker React Component

The React wrapper is not available yet. If you need it, please open an issue in [`jb-time-picker`](https://github.com/javadbat/jb-time-picker/issues).

For current usage, use the web component directly:

```js
import 'jb-time-picker';
```

```html
<jb-time-picker></jb-time-picker>
```

## Shared Documentation

For web-component behavior, events, slots, and CSS variables, see [`jb-time-picker`](https://github.com/javadbat/jb-time-picker).
